## Critical Test Cases Included

### Edge Cases for psi
- **psi = 0.0**: Boundary exactly at grid point (Dirichlet)
- **psi = 0.001**: Nearly at grid point (tests numerical stability)
- **psi = 0.1, 0.25**: Small offsets
- **psi = 0.5**: Midpoint
- **psi = 0.75, 0.9**: Near next grid point
- **psi = 1.0**: Exactly at next grid point (Floating)

### Both Ray Directions
- **ray_outside = false**: Pointing into domain (left boundary)
- **ray_outside = true**: Pointing out of domain (right boundary)

### Multiple Grid Spacings
- **h = 0.1**: Fine grid
- **h = 0.5**: Medium grid
- **h = 1.0**: Unit spacing (common in tests)
- **h = 2.0**: Coarse grid

## Validation Tolerances

Based on double precision (IEEE 754):

| Comparison Type | Recommended rtol | Notes |
|----------------|------------------|-------|
| Interior stencils | 1e-14 | Should be bit-exact |
| Simple boundary stencils | 1e-14 | Should be bit-exact |
| Polynomial boundary stencils | 1e-12 | May have minor differences |
| Different algorithms | 1e-6 to 1e-10 | Document differences |

### Test Structure

# tests/test_stencil_e2_1.py
import pytest
from cpp_reference import load_reference_data

ref_data = load_reference_data()

class TestE2_1Interior:
    @pytest.mark.parametrize("h", [0.1, 0.5, 1.0, 2.0])
    def test_interior(self, h):
        from stencil import E2_1
        stencil = E2_1(alpha=[1, 2, 3, -1])

        computed = stencil.interior_coefficients(h)
        reference = ref_data.get_interior("E2_1", h)

        np.testing.assert_allclose(computed, reference, rtol=1e-14)

class TestE2_1FloatingBoundary:
    @pytest.mark.parametrize("psi", [0.1, 0.25, 0.5, 0.75, 0.9, 1.0])
    @pytest.mark.parametrize("ray_outside", [False, True])
    def test_floating(self, psi, ray_outside):
        from stencil import E2_1
        stencil = E2_1(alpha=[1, 2, 3, -1])

        h = 1.0
        computed = stencil.nbs_coefficients(h, psi, "floating", ray_outside)
        reference, r, t = ref_data.get_boundary(
            "E2_1", "floating", h, psi, ray_outside
        )

        np.testing.assert_allclose(computed, reference, rtol=1e-14)
